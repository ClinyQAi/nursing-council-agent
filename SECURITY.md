# Security Policy

## Supported Versions

The following versions of the Nursing Council Agent are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | ✅ Yes             |
| 1.x.x   | ❌ No              |

## Reporting a Vulnerability

As this project is designed for use in Nursing Education and Healthcare environments, we take security very seriously. 

**PLEASE DO NOT USE PUBLIC ISSUES TO REPORT SECURITY VULNERABILITIES.**

If you discover a potential security vulnerability, please report it privately:

1.  **Email**: Send a detailed report to **info@nursingcitizendevelopment.com**.
2.  **Contents**: Please include a description of the vulnerability, steps to reproduce, and any potential impact on sensitive data.

### Our Commitment
*   We will acknowledge receipt of your report within 48 hours.
*   We will provide a timeline for a fix and keep you updated on progress.
*   We will coordinate a public disclosure if necessary, once a fix is available.

---

**Note**: This project does not store Patient Data (GDPR/HIPAA). However, protecting user API keys and institutional credentials is our top priority.
